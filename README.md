# Unity Challenge Walkthrough 
In this challenge, you'll apply the skills you learned while making your VR Room in an industrial training simulation. In this app, using buttons, knobs, levers, and joysticks, the user has to collect crates dropped into the factory and stack them on a nearby platform.

# MurataVR_Safety_Hazard_Training
The simulation will immerse Murata employees in realistic fire emergency scenarios within a factory setting, equipping them with the necessary decision-making and motor skills to respond effectively to electrical fire incidents.
